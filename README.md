# C++ Programming Showcase: Integrated Console Hub

## 🕹️ Project Overview
This repository serves as a portfolio of my C++ programming fundamentals. It integrates various exercises and logic tests into a single, modular console application managed by a structured menu system.

The project demonstrates the practical application of basic programming concepts in a clean and organized manner.

## 🚀 Features
- **Mini Slot Machine**: A simulation demonstrating random number generation and logic verification for winning combinations.
- **Guess the Number**: An interactive game utilizing loops, conditional statements, and user feedback.
- **Prime Number Tester**: A mathematical tool implemented with an optimized algorithm for efficiency.

## 🛠️ Technical Skills Demonstrated
- **Modular Code Structure**: Separation of logic into reusable functions.
- **State Management**: Using `switch-case` for application navigation.
- **Input Validation**: Handling `std::cin` errors to ensure program stability.
- **Cross-Platform Handling**: Console management adapted for both Windows and Unix-like systems.

# Portfolio C++: Zintegrowany Hub Konsolowy

## 🕹️ Przegląd projektu
To repozytorium stanowi portfolio moich podstaw programowania w języku C++. Zamiast publikować zbiór luźnych plików, zintegrowałem swoje ćwiczenia i zadania w jedną, modułową aplikację konsolową, zarządzaną przez przejrzysty system menu.

Projekt został stworzony z naciskiem na **czystość kodu**, **modułowość** oraz **praktyczną implementację logiki** — kluczowe umiejętności w codziennej pracy programisty.

## 🚀 Funkcje programu

### 1. Mini Automat (Slot Machine)
Symulacja mechaniki prostego automatu do gier. Demonstracja:
- Generowania stanów losowych przy użyciu `rand()` i `srand()`.
- Logiki weryfikacji kombinacji wygrywających (Jackpot vs. Mała wygrana).
- Przepływu interakcji z użytkownikiem.

### 2. Gra: Zgadnij Liczbę
Klasyczna gra logiczna, w której wykorzystano:
- Pętle sterujące oraz instrukcje warunkowe.
- Mechanizm interakcji zwrotnej (podpowiedzi "za dużo" / "za mało").

### 3. Zoptymalizowany Tester Liczb Pierwszych
Narzędzie matematyczne do sprawdzania, czy liczba jest pierwsza, wykorzystujące:
- Optymalizację algorytmiczną **6k±1**, która znacząco ogranicza liczbę iteracji pętli.
- Dowód dbałości o wydajność i wydolność algorytmów.

## 🛠️ Zaprezentowane umiejętności techniczne
- **Zarządzanie stanem aplikacji:** Wykorzystanie struktur `switch-case` do nawigacji po menu.
- **Programowanie modułowe:** Podział kodu na funkcje, co ułatwia czytelność i utrzymanie projektu.
- **Walidacja danych:** Obsługa błędów strumienia `std::cin`, co zapobiega awariom programu przy błędnym wprowadzeniu danych (np. litery zamiast cyfry).
- **Obsługa wieloplatformowa:** Zastąpienie komend specyficznych dla Windows (jak `system("PAUSE")`) standardowymi rozwiązaniami biblioteki C++.
